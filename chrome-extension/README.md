# Draw.io Boolean Operations Chrome Extension

## Install

1. Open `chrome://extensions/` in Chrome.
2. Enable `Developer mode` in the upper-right corner.
3. Click `Load unpacked`.
4. Select this `chrome-extension` directory.
5. Close all open `app.diagrams.net` tabs and reopen the editor.

The commands appear under `Arrange > Boolean Operations`.

## Scope

The extension runs only on `https://app.diagrams.net/*`. Paper.js and the
Boolean Operations plugin are bundled locally, so the extension does not
download executable code at runtime.
